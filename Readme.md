# Employee Management System using MongoDB

This repository contains the assignment for an Employee Management System to practice basic MongoDB operations.

## Overview
This assignment focuses on performing basic CRUD operations (Create, Read, Update, Delete) and using query operators. It simulates an HR department's Employee Management System using a NoSQL document-oriented approach.

## Operations Covered
- **Database & Collection Setup**: Creating and switching databases (`employeeDB`).
- **Insert Operations**: Inserting single and multiple employee records.
- **Read Operations**: Querying documents using various operators.
- **Update Operations**: Modifying existing employee records.
- **Delete Operations**: Removing documents from the collection.

## Files
- `Employee_MongoDB_Assignment.docx`: The assignment document containing instructions and queries.

## Setup
1. Start the MongoDB shell by running `mongosh`.
2. Use the database by running `use employeeDB`.
3. Follow the instructions in the `.docx` file to practice the CRUD operations.
